1. Clone the CAP05_CHALLENGE_InternetWhisper repository:
```sh
❯ git clone https://github.com/Mvargas08/CAP05_CHALLENGE_InternetWhisper
```

2. Navigate to the project directory:
```sh
❯ cd CAP05_CHALLENGE_InternetWhisper
```
